# Changelog

## [1.1] - 2026-07-15

### Fixed
- **Admin Panel Scrolling**: Added `max-height: 85vh` and `overflow-y: auto` to modal content, allowing full interaction with all admin panel elements

### Changed - Color Scheme Update

#### Sidebar
- **Old**: Solid Navy (#00205B)
- **New**: Dark gradient (`#1a1a2e → #16213e → #0f1419`)
- More depth and visual interest while maintaining readability

#### Primary Actions & ESP Selector
- **Old**: Teal (#72D1C8)
- **New**: Orange (#FDB768)
- Matches the "In Progress" tag color from brand guidelines
- Active ESP button now uses orange with dark text for better contrast

#### Buttons & Interactive Elements
- Send button: Orange with dark text
- Submit buttons: Orange with dark text
- ESP selector active state: Orange background
- Focus states: Orange borders
- Loading indicator dots: Orange

#### Welcome Message
- **Old**: Lime → Teal → Coral gradient
- **New**: Orange gradient (`#FDB768 → #F8937D → #F5a84d`)
- Text color changed to dark (#1a1a2e) for better contrast

#### User Messages
- **Old**: Solid Navy
- **New**: Dark gradient matching sidebar

### Visual Summary

**Before:**
- Navy sidebar (solid)
- Teal buttons and active states
- Lime accents

**After:**
- Dark gradient sidebar (1a1a2e → 16213e → 0f1419)
- Orange buttons and active states (#FDB768)
- Orange accents throughout
- Better contrast and readability
- Scrollable admin panel

All changes maintain the Yotpo brand identity while improving usability and visual appeal.
