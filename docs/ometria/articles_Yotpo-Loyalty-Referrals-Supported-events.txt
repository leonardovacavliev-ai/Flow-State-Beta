These are the custom events Ometria supports for our integration with
Yotpo Loyalty & Referrals
.
You can use these events to
trigger automation campaigns
.See:
Yotpo Loyalty & Referrals: Use cases
All of these events also contain <strong>core fields
you can use in the
customer filter
to create segments or as merge tags</a> in your marketing templates.
Every event has one field thatcontains the full Yotpo payload.
You can only use the contents of this field in
merge tags
.
To create amerge tag for any fields within that payload (i.e. fields that arent already part of the core fields), please see the link to Yotpos documentation in each table below to view the payload structure. yotpo_loyalty_opt_in

An opted-in customer can earn points, join tiers, redeem points, etc.
