Shopify, Shopify Plus, BigCommerce, Salesforce Commerce Cloud, Adobe Commerce (Magento)

The Ometria integration with Yotpo Loyalty & Referrals enables you to store your Yotpo loyalty data against your Ometria contact and product records.

This integration automatically imports your live Loyalty contact data into Ometria, so you can personalize your campaigns and uncover new insights to inform your marketing strategy.

### About Ometria

Ometria combines the data capabilities of a CDP with a cross-channel marketing orchestration platform, letting retail marketers create relevant experiences throughout the customer journey.

### Before you begin

You’ll first need to do the following:

- Create an API key in Ometria. To learn how, refer to theOmetria help center

- Find your Yotpo API key and GUID. To learn how, seeFinding your Loyalty & Referrals API Key and GUID

### Setting up the integration

1. In Ometria, go to Settings and select theConnectionstab.

2. ClickAdd new connection, then selectYotpo loyalty & referrals. The Ometria authentication screen displays.

3. ClickNew authentication.

4. Enter the API key you created at the beginning of the process and selectCreate.

5. ClickNew authentication.

6. Change your authentication name if you wish. This name will help you identify any other Yotpo accounts you might have added in Ometria.

7. Enter your Yotpo API key and GUID, and clickCreate.

8. Once the authentication is successful, clickNextto reach the Yotpo Loyalty & Referrals webhook screen.

9. Copy the Webhook URL and log into your Yotpo Loyalty & Referrals account.

10. In Yotpo, go toSettings > General Settings, and paste the URL into the Yotpo Webhook URL field.Tip:If you already have a webhook registered, you can add this one in the same field by separating the URLs using semicolons (;).

11. Save these changes in Yotpo, then return to Ometria.

12. SelectFinish. Your Yotpo Loyalty & Referrals connection is set up.

Now you’ve set up the integration, you’ll need to create your custom fields. To learn how, and to see some use cases for this integration, refer to theOmetria Loyalty & Referrals integration page.
