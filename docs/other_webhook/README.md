# Other/Webhook ESP Documentation

This directory contains documentation for generic ESP platforms that use webhook-based integrations with Yotpo Loyalty.

## Adding Documentation

To add documentation URLs for webhook-based integrations:

1. Go to the Admin panel (password: RICHCSM)
2. Find the "Other/Webhook" section
3. Enter the documentation URL in the "Add new link" field
4. Click "Add Link"
5. Click "Refresh All" to crawl and vectorize the new documentation
